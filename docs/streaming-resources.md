## Overview
Stremio is a free streaming video application for watching movies, TV series, live TV, sports, and more.

## Features
*   **Freedom to Stream:** Enjoyed by more than 30 million users worldwide.
*   **Streaming Enhanced:** Offers a secure, modern, and seamless entertainment experience with 4K HDR support.
*   **Easy to Use:** Clean and intuitive interface.
*   **Seamless Across Devices:** Available for Windows, Mac, Linux, Android, Android TV, and via web browser. Keeps track of your progress across all devices when logged in.
*   **Open Source:** The code is publicly auditable.
*   **Privacy Focused:** Respects user privacy, collecting only essential data. Offers a Guest mode requiring no data.
*   **Casting:** Supports Chromecast from desktop and mobile apps.
*   **Extensible:** Can be extended beyond the official addon catalog.
*   **Magnet/Torrent Support:** Can play magnet links, HTTP links, and torrent files.
*   **Offline Playback:** (Upcoming feature) Can cache videos for offline viewing.
