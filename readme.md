## üöÄ Quick Start Guides
*Direct links to the most frequently used procedures.*

### 1. New Android TV Box Checklist
1.  **Enable Developer Options:** Go to `Settings > About > Build Number` (Click 7 times).
2.  **Sideloading:** Install [Downloader App](www.aftvnews.com) to get APKs not in the Play Store.
3.  **App Setup:** Use the [Essential Apps List](./streaming-resources/popular-apps.md) to populate your home screen.

### 2. The Kodi 5-Minute Setup
1.  Download the latest ARMv8 (64-bit) APK from the [Official Kodi Site](kodi.tv).
2.  Follow the [Subtitle Configuration Guide](./kodi/subtitle-setup.md) to ensure Asian fonts display correctly.
3.  Install the **C-Drama Add-on** using the steps in [Add-ons List](./kodi/addons-list.md).

### 3. Watching Chinese Dramas (C-Dramas)
*   **Best Free Experience:** Check the [Verified Web Directory](./streaming-resources/sites-directory.md) for sites with minimal ads.
*   **Best Paid Experience:** Comparison of VIP features for 2025 is in [Streaming Resources](./streaming-resources/chinese-dramas.md).
